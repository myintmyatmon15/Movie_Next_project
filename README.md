# CosmoFlix

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## 🚀 About CosmoFlix

**CosmoFlix** is a modern, high-performance movie discovery platform heavily inspired by the sleek design of Plex. This web application allows users to seamlessly explore movies across various genres with a fully responsive and visually stunning user interface.

### ✨ Features
* **Plex-Inspired UI:** Features a beautiful dark-themed layout with responsive glassmorphism navigation and components.
* **Dynamic Routing (Next.js 15+):** Optimized routing structures utilizing `gener/[name]/[id]` patterns for seamless category navigation.
* **TMDB API Integration:** Fetches real-time movie lists, genres, and metadata directly from The Movie Database.
* **Fully Responsive:** Designed using a Tailwind CSS Grid system to ensure a perfect viewing experience across all devices (Mobile, Tablet, Desktop).

### 🛠️ Tech Stack
* **Framework:** Next.js 15+ (App Router)
* **Styling:** Tailwind CSS, Lucide React (Icons)
* **Language:** TypeScript
* **Data Source:** TMDB API
* **Deployment:** Vercel
